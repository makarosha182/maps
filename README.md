# Сервис-советник по Сивасу 🏛️

AI-powered туристический советник для города Сивас, Турция, построенный на базе Claude API и веб-скрапинга официального туристического сайта.

## 🌟 Особенности

- **Полный веб-скрапинг** официального сайта sivas.goturkiye.com
- **Векторная база данных** для семантического поиска контента
- **Интеграция с Claude API** для генерации интеллектуальных ответов
- **Веб-интерфейс** с современным дизайном на русском языке
- **Поддержка вопросов** о достопримечательностях, кухне, культуре, истории
- **Источники информации** для каждого ответа

## 🚀 Быстрый старт

### 1. Клонирование и установка

```bash
# Создание виртуального окружения
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows

# Установка зависимостей
pip install -r requirements.txt
```

### 2. Настройка API ключа

Создайте файл `.env` из примера:
```bash
cp env_example.txt .env
```

Отредактируйте `.env` и добавьте ваш Claude API ключ:
```
CLAUDE_API_KEY=your_actual_claude_api_key_here
```

### 3. Автоматическая настройка

```bash
python setup.py
```

Этот скрипт автоматически:
- Скачает все страницы сайта Сиваса
- Создаст векторную базу данных
- Протестирует интеграцию с Claude
- Подготовит сервис к запуску

### 4. Запуск сервиса

```bash
python main.py
```

Откройте браузер и перейдите на http://localhost:8000

## 🔧 Компоненты системы

### Веб-скрапер (`scraper.py`)
- Использует Selenium + requests для скачивания контента
- Поддерживает динамический контент JavaScript
- Извлекает заголовки, текст, изображения, ссылки
- Сохраняет структурированные данные в JSON

### Векторная база (`vector_store.py`)
- ChromaDB для хранения эмбеддингов
- Sentence Transformers для генерации векторов
- Семантический поиск по содержимому
- Классификация типов страниц (еда, достопримечательности, культура)

### Claude сервис (`claude_service.py`)
- Интеграция с Anthropic Claude API
- Русскоязычные промпты и ответы
- Контекстные ответы на основе найденного контента
- Предложения популярных вопросов

### Веб-интерфейс (`main.py`, `templates/`, `static/`)
- FastAPI backend с REST API
- Современный responsive интерфейс
- Чат-интерфейс в реальном времени
- Отображение источников информации

## 📊 API Endpoints

- `GET /` - Главная страница с чат-интерфейсом
- `POST /api/ask` - Задать вопрос советнику
- `GET /api/suggestions` - Получить примеры вопросов
- `GET /api/health` - Проверка состояния сервиса

## ⚙️ Настройки

Основные параметры в `config.py`:
- `MAX_PAGES` - максимум страниц для скрапинга
- `CHUNK_SIZE` - размер фрагментов для векторизации
- `API_PORT` - порт веб-сервиса
- `DELAY_BETWEEN_REQUESTS` - задержка между запросами
