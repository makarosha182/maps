# 🎉 СЕРВИС ГОТОВ! - Инструкции по использованию

## ✅ ЧТО СЕЙЧАС РАБОТАЕТ

### 🌐 Демо веб-сервер (АКТИВЕН)
**URL: http://localhost:8001**

Сейчас запущен демонстрационный сервер, который:
- ✅ Показывает красивый веб-интерфейс
- ✅ Выполняет семантический поиск по контенту Сиваса  
- ✅ Возвращает релевантные результаты
- ✅ Работает на русском языке
- ⚠️ Без Claude API (только векторный поиск)

### 📊 Данные обработаны:
- **10 страниц** сайта Сиваса
- **6,654 слов** контента
- **60 чанков** в векторной БД
- **9 категорий** (достопримечательности, еда, культура и т.д.)

## 🚀 КАК ИСПОЛЬЗОВАТЬ

### 1. Веб-интерфейс
1. Откройте браузер
2. Перейдите на **http://localhost:8001**
3. Задавайте вопросы о Сивасе в чат-интерфейсе

### 2. API напрямую
```bash
# Пример запроса
curl -X POST "http://localhost:8001/api/ask" \
  -H "Content-Type: application/json" \
  -d '{"query": "Какие достопримечательности есть в Сивасе?"}'
```

### 3. Примеры вопросов
- "Какие достопримечательности есть в Сивасе?"
- "Что можно посмотреть за 48 часов?"
- "Расскажите об истории Сиваса"
- "Какие активности доступны туристам?"
- "Что попробовать из местной кухни?"

## 🔧 НАСТРОЙКА CLAUDE API (ОПЦИОНАЛЬНО)

Если хотите полноценные AI-ответы вместо простого поиска:

1. **Проверьте Claude API ключ:**
   ```bash
   # Замените на действующий ключ
   echo "CLAUDE_API_KEY=sk-ant-api03-новый_ключ" > .env
   ```

2. **Запустите полную версию:**
   ```bash
   pkill -f "simple_demo_server"  # Остановить демо
   source venv/bin/activate
   python main.py  # Запустить полную версию на :8000
   ```
